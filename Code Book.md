Subject_id 
    Identifies the subject performing the observation
    
    Integer: 1- or 2- digit
    
    Range: 1:30
    
Activity
    Brief description of the activity performed during the observation
    Character: 7-18 characters long
    WALKING
    WALKING_UPSTAIRS
    WALKING_DOWNSTAIRS
    SITTING
    STANDING
    LAYING
